INSTRUCTIONS FOR CODE SETUP!!!! IMPORTANT!!! FOLLOW THESE!!!!!!!!!!

1. Go to https://groq.com/
2. Click Start Building
3. Make an account
4. Click API Keys in the top right
5. Click create API key, give it any name, and an unlimited expiration date
6. DON'T CLOSE THE POPUP! YOU WILL NEED THE API KEY!!
7. Go to wit.ai
8. Create account/login
9. Click New App, name it anything
10. Go to Management -> Settings
11. DON'T CLOSE THIS EITHER! YOU WILL NEED THE KEY!!
12. Download the main zip. Code -> Download ZIP
13. Open it in your file explorer, extract, navigate to Downloads\SunglassesofPain-main\SunglassesofPain-main\dumbsunglasses\dumbsunglasses.ino and double click. It should open in Arduino IDE.
14. Click on keys.h.
15. <img width="398" height="103" alt="image" src="https://github.com/user-attachments/assets/7829e22b-1cbc-428b-a673-09fe24170f43" />
16. Update the WiFi SSID and PASS to your settings.
17. Update the Groq and Wit API keys with the ones you just made.
18. Click File -> Preferences -> Additional board manager URLs, and enter this https://espressif.github.io/arduino-esp32/package_esp32_index.json
19. Click the Board manager <img width="65" height="57" alt="image" src="https://github.com/user-attachments/assets/b88ce9b5-29b9-4c69-873d-0499d45dc04d" />
20. Search for "ESP32" by Espressif Systems
21. Click install and wait for it to finish
22. For every one of the ZIPs in Downloads\SunglassesofPain-main\SunglassesofPain-main do this
    
- Click Sketch -> Include Library -> Add .zip library -> Select the zip from Downloads\SunglassesofPain-main\SunglassesofPain-main
   <img width="411" height="338" alt="image" src="https://github.com/user-attachments/assets/8c7af526-908d-48ef-ba49-1953638fa6b2" />
- Wait for it to process
